# DEFINITION OF TERMS

The following terms are defined operationally according to their specific application in this research, arranged in alphabetical order:

* **3D Spatial Coordinate Rotation** – A mathematical transformation algorithm that utilizes live IMU pitch values to align camera-centric depth data with the world gravity vector, ensuring physical distance accuracy regardless of the teacher’s chest-tilt angle.
* **Behavior Status Tracking** – A real-time computer vision capability that identifies and logs students' active classroom participation states, such as hand-raising or attentiveness, to provide situational awareness to the visually impaired teacher.
* **Blind Assistive Device** – A wearable technological system that utilizes artificial intelligence and spatial sensors to provide non-visual environmental feedback, assisting individuals with visual impairments in navigation and daily tasks.
* **Claude 4.5 Haiku API** – A cloud-linked vision-language model that processes visual images to generate descriptive, natural-language summaries of the classroom environment for context-aware assistance.
* **Confidence Accumulator** – A temporal software smoothing filter that utilizes a rolling frame window and majority-vote logic to stabilize student posture and behavior classifications, eliminating split-second visual flickering.
* **Face Blurring** – A privacy-preserving visual algorithm that dynamically identifies facial regions using human pose keypoints and applies a blurring mask to anonymize students on the monitoring interface.
* **Mobility Efficiency** – A navigation performance metric that measures the optimization of a user’s travel speed and path-finding success when traversing a cluttered environment using the SENSEY system.
* **Multimodal Feedback** – A sensory communication method that utilizes a combination of distinct output channels, specifically spoken natural language and spatial audio sound alerts, to convey information to the user.
* **NPU-Accelerated** – A hardware-enhanced processing state that offloads deep learning and computer vision computations from the main CPU to a dedicated neural processing core to achieve high real-time frame rates.
* **Walkable Path** – A spatial navigation concept that represents an unobstructed physical lane detected by the system that is wide enough for the visually impaired teacher to traverse safely.
* **YOLOv8m Object Detection** – A real-time deep learning algorithm that identifies and isolates pre-trained visual categories, such as chairs, desks, and backpacks, to locate obstacles in the teacher's navigation path.
* **YOLOv8m-pose Estimation** – A deep-learning pose estimation algorithm that detects seventeen human joint keypoints to map body geometry and classify postural states.

* **Walkable Path** is defined as a *"spatial navigation concept,"* preventing any repetition of "walkable" or "path."
